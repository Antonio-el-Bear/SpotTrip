# SpotTrip Minimal Project

This is a minimal version of the SpotTrip project, containing only the essential files for debugging and collaboration.

## Structure
- `backend/` - Django backend (core logic, models, views, serializers)
- `tourguide-frontend-fresh/src/` - Svelte frontend source code
- `requirements.txt` - Python dependencies
- `package.json` - Frontend dependencies

## Excluded
- All build artifacts, cache, and environment folders (e.g., `node_modules`, `build`, `static`, `venv`, `__pycache__`, `.vscode`, `db.sqlite3`)

## Usage
1. Clone this repository
2. Install backend and frontend dependencies
3. Run backend and frontend servers as described in their respective README files
